# CueBoard Features

## Wat is CueBoard?
CueBoard is een Show OS voor live evenementen. Het helpt teams om shows, presentaties en evenementen professioneel te runnen met draaiboeken, timers, en real-time samenwerking.

## Kernfuncties

### Draaiboeken (Rundowns)
- Maak draaiboeken met cues (items) per show
- Cue types: video, audio, lighting, speech, break, custom, intro, outro, presentation
- Drag-and-drop herordenen
- Timer per cue met countdown
- Auto-advance naar volgende cue
- Kleurcodes voor visueel onderscheid
- Technische notities (alleen voor technici zichtbaar)
- Spreker/presentator toewijzen per cue

### Live Show Views
- **Caller View** — De regisseur/stage manager bedient de show, start/stopt cues
- **Crew View** — Read-only overzicht voor technici
- **Presenter View** — Teleprompter/timer voor sprekers (met PIN-beveiliging)
- **Output View** — Scherm voor op het podium (slides + timer)
- **Clock View** — Grote timer voor op een scherm

### Team Samenwerking
- Rollen: owner, editor, caller, crew, presenter, viewer
- Real-time synchronisatie via Supabase Realtime
- In-app chat per draaiboek

### Cast Portal
- Token-based toegang voor cast/gasten
- Geen account nodig — link delen volstaat
- Zien hun eigen cues en timing

### Green Room
- Aparte ruimte voor communicatie met gasten

### Companion Integratie
- Bitfocus Companion koppeling
- Hardware buttons om cues te triggeren
- Webhook-based communicatie

### Media
- Audio/video uploads per cue
- PDF/PPTX presentatie-viewer
- Slide control (caller, presenter, of both)
- Still image als achtergrond tussen sprekers

### Mic Patch Panel
- Audio device mapping (handheld, headset, lapel, table, IEM)
- Kanaal toewijzing per cue (before, during, after)

### Overig
- Callsheet printen
- Draaiboek exporteren
- Draaiboek templates
- Push notificaties
- Public status screen (zonder login)
- Zoekfunctie
